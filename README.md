# doctorai_gpt3_aws

Set up an r4.large instance on EC2 with Neo4j community running.

## build
sam build

## deployment
sam deploy --guided --capabilities CAPABILITY_NAMED_IAM

## destruction
sam delete --stack-name doctorai
